JSLibrary

Description

JSLibrary is a library of Java classes available for use, for free, in any program. It started off as a replacement for the Park library, but has since added extra classes not found in Park. The library in its current form contains classes related to interface elements, dynamic data structures and utility methods.

Currently included in JSLibrary:

  • JSFrame and JSDialog, providing animation for JFrame and JDialog
  • JSBadgeButton, an extension to JButton providing iOS style notification badges
  • JSPreferenceWindow, an API for creating Mac OS X style Preferences windows
  • JSPrintPreview, an alternative to PC_Paper for creating print preview windows
  • JSUtil, a class containing many helpful methods, such as formatting phone numbers and address lookups
  • ... plus more!
